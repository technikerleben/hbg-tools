Phase 1

Erst ohne Computer und KI planen – dann programmieren!

Arbeitsblatt · Kleingruppen · Schreibplan

E – Eingabe

  • Monatliches Einkommen
  • Miete / WG-Miete
  • Lebensmittel
  • Freizeit
  • Transport (Auswahl)
  • Wohnnebenkosten
  • Eigene Ideen?

V – Verarbeitung

  • Ausgaben addieren
  • WG: Miete ÷ Personen
  • Restbetrag berechnen
  • Budget bewerten

A – Ausgabe

  • Gesamtausgaben
  • Restbetrag
  • Bewertung:
    reicht / knapp / reicht nicht
📋 Pflichtbestandteile
  • Einkommen (Startwert 1.100 €)
  • Miete oder WG-Miete
  • Lebensmittel
  • Freizeit
  • Transport (Fahrrad / ÖPNV / Roller / Auto)
  • Wohnnebenkosten (160 € pauschal)
✍️ Aufgaben für die Gruppe
  • Eingaben festlegen
  • Verarbeitung beschreiben
  • Ausgabe beschreiben
  • Designwünsche notieren
  • Ersten Prompt formulieren
Tipp

So sieht ein guter Prompt aus

Stichwortlisten reichen – kein langer Text nötig!

# Budget-Simulator als Single-File-HTML Thema: Meine erste eigene Wohnung Eingabe: - Einkommen: 1.100 € (veränderbar) - Miete: 420 € (veränderbar) - WG-Miete: Gesamtmiete + Anzahl Personen - Lebensmittel: 220 € - Freizeit: 100 € - Transport: Fahrrad / ÖPNV / Roller / Auto - Wohnnebenkosten: 160 € (pauschal) Verarbeitung: - Alle Ausgaben addieren - WG: Miete ÷ Personen - Restbetrag = Einkommen − Ausgaben Ausgabe: - Gesamtausgaben, Restbetrag - Bewertung: reicht / knapp / reicht nicht Design: - Große Felder, iPad-tauglich, modern # Alles in einer einzigen HTML-Datei!
💡 Das Geheimnis

Die KI versteht strukturierte Stichpunkte genauso gut wie langen Text – und du sparst Zeit beim Schreiben.

Reihenfolge:
1. Aufgabe benennen
2. Eingaben listen
3. Berechnung erklären
4. Ausgabe beschreiben
5. Design-Wünsche ergänzen
⚠ Denk dran:
"Schreibe den gesamten Code in eine einzige HTML-Datei" – immer am Ende ergänzen!
Phase 2

Am Computer – los geht's!

Informatikraum · ais.chat · HTML testen

1
Anmelden Am Computer im Informatikraum mit IServ anmelden · Die Aufgabe enthält den Link zu ais.chat
2
Prompt eingeben Euren vorbereiteten Prompt aus dem Arbeitsblatt in ais.chat einfügen und abschicken
3
Code speichern Den HTML-Code kopieren und auf code-zu-html.vercel.app einfügen → als HTML-Datei herunterladen
4
Testen Alle Eingaben verändern, Berechnungen prüfen, Design beurteilen
5
Verbessern Einen konkreten Verbesserungswunsch formulieren und erneut an ais.chat schicken
✅ Testen: Das muss funktionieren
  • Alle Eingaben veränderbar
  • Berechnung nachvollziehbar
  • Ausgabe verständlich
  • Auf dem iPad gut nutzbar

📈 Verbesserungswunsch

Notiert auf dem Arbeitsblatt einen konkreten Verbesserungswunsch – je genauer, desto besser die KI-Antwort!

„Die WG-Berechnung soll deutlicher erklärt werden." ✓
„Mach es schöner." ✗

  • Berechnung ist korrekt
  • Bewertung wird angezeigt
